# SGPA
SGPA est une application desktop de gestion de pharmacie développée en Java/JavaFX avec Maven et MySQL. Elle permet de gérer les médicaments (lots/péremption/stock), les ventes, les fournisseurs, les commandes, les utilisateurs/roles, les rapports financiers (HT/TVA/TTC/marge) et l’export PDF/Excel (facture, devis, rapports).
